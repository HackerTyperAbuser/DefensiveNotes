NTFS Index Attribute is another artifact of the Windows filesystems to track which files are in which directories. After each folder update (delete a file), this index will be updated to reflect new changes. A track of deleted files can still be tracked even if securely wiped. 
- Best way to prove existence of a deleted file.
- Each directory has a seperate \$I30 file
- Tools: `MFTEcmd` or `INDXRipper`
